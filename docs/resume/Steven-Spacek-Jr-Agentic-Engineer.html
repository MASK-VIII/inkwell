Featured Project — Inkwell

Inkwell — Local-first writing workspace

Founder / builder · Cursor (agentic development) · 2025–2026 · ~99 commits shipped to production

  • Product: Chapter-first manuscripts, linked notes, print & ebook formatting, optional cloud library backup; marketing/SEO and tiered subscriptions (Basic/Pro).
  • Cloud sync: Supabase Storage for library zips, library_heads revision table, RPC push with optimistic concurrency and explicit conflict resolution (keep device / use cloud / download both).
  • Desktop: Windows/macOS Electron app with secure auth storage, deep links, auto-update via GitHub Releases, and CI that validates cloud credentials in production bundles.
  • Print/ebook engine: Shared TipTap mark model, worker-based spine pagination, PDF preflight tests, documented parity matrix for maintainability.
  • Monetization: Paddle Billing checkout and signed webhooks → user_entitlements with tier-aware storage policies.

Stack: TypeScript, React, Vite, Electron, Supabase, Paddle, Vitest, GitHub Actions
